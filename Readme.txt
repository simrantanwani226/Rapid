###College Comparer & Ranking Website###
##Overview##
This project aims to develop a college comparer and a static website to showcase the ranking of particular colleges. The website provides users with the ability to compare different colleges based on various criteria such as location, programs offered, faculty, infrastructure, and more. Additionally, the website features a ranking system that highlights top-performing colleges based on predefined metrics.

##Features##
College Comparison: Users can compare multiple colleges side by side based on different parameters.
Search Functionality: Users can search for colleges based on specific criteria or keywords.
Detailed College Profiles: Each college has a dedicated profile page providing comprehensive information about the institution.
Ranking System: The website ranks colleges based on predetermined factors such as academic excellence, student satisfaction, placement records, etc.
Responsive Design: The website is designed to be accessible and user-friendly across various devices and screen sizes.
##Technologies Used##
Frontend: HTML, CSS, JavaScript
Backend: php
Version Control: Git

Open the index.html file in your preferred web browser.
##Usage##
Browse through the list of colleges or use the search functionality to find specific institutions.
Select colleges for comparison and view detailed comparisons.
Explore individual college profiles for more information.
Check the rankings to see top-performing colleges.
